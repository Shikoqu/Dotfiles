## Recomendations

AeroSpace guide and goodie provide collection of pretty nice settings to enable so the expeirience is smooth and enjoyable.

### 1. Group windows by application in Mission Controll

   Settings -> Desktop & Dock -> Mission Control -> Group windows by application

### 2. Disable Displays have separate Spaces

   Settings -> Desktop & Dock -> Mission Control -> Displays have separate Spacesa

### 3. Move windows by dragging any part of the window

   I wasn't able to make it work but here is the command:  
   `defaults write -g NSWindowShouldDragOnGesture -bool true`  
   I think the issue lays in clicking while holding option which triggers right click.

### 4. Highlight focused windows with colored borders

   using FelixKratz JankyBorders

### 5. AeroSpace Raycast extension

### 6. Use trackpad gestures to switch workspaces

   [SwipeAeroSpace](https://github.com/MediosZ/SwipeAeroSpace  )

### 7. Disable annoying and useless "hide application" shortcut

   already in `aerospace.toml` settings
